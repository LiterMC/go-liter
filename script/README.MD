# TS support

1. Install the package below:
   ```sh
   npm install --save-dev go-liter-plugin
   ```

2. Add the following element to your `tsconfig.json`
   ```js
   {
   "compilerOptions": {
   		"typeRoots": [
   			"node_modules/@types",
   			"node_modules/go-liter-plugin"
   		]
   	}
   }
   ```
